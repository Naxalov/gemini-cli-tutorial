# Gemini CLI Tutorial Context

## Directory Overview
This directory serves as a documentation hub for the Gemini Command Line Interface (CLI). It contains tutorials, installation guides, and usage examples designed to help users get started with and master the Gemini CLI tool.

## Key Files
*   **`README.md`**: The primary documentation file. It provides:
    *   An introduction to the project.
    *   Installation instructions (specifically for macOS using Homebrew).
    *   A high-level overview of command types (CLI Arguments vs. Slash Commands).
    *   A reference for basic commands (Configuration, History, Interaction).
    *   A list of Slash Commands (`/help`, `/bug`, `/clear`, `/exit`).
    *   Advanced usage details, including **Custom Commands** for creating prompt shortcuts.
*   **`getting-started.md`**: A step-by-step guide for new users. It covers:
    *   Prerequisites.
    *   Obtaining a Gemini API key.
    *   Configuring the CLI with the API key.
    *   Running the first command (including the `--prompt` flag).
    *   Checking command history.

## Usage
The content in this directory is intended to be read by users who want to learn how to use the Gemini CLI.
1.  **Installation**: Users start with `README.md` to install the tool.
2.  **Configuration**: Users follow `getting-started.md` to set up their API key.
3.  **Reference**: Users refer back to `README.md` for a list of available commands and syntax.
